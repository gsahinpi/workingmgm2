Magmio Documentation

Doxygen Documentation overview

Magmio Doxygen Documentation describes Application Programming Interface (API) from the user's perspective. There are described classes, methods, data structures and enumeration types that used in the API. There are examples enclosed that show how to use these classes and methods to control whole trading solution.

When learning how to use Magmio solution, it is recommended to start with Magmio User's Manual and go through Doxygen documentation afterwards. When a class or a method is mentioned in User's Manual, it contains a link to appropriate page in Doxygen Documentation.

Brief description of important classes

Magmio class

Magmio class covers whole API. All other classes are initiated inside of this class. User's application has to access to all API methods through Magmio class and to all data structures through Magmio namespace.

Parameter classes

All parameters should be accessed through their appropriate classes. There are GlobalParams class to access Global parameters, PerGroupParams class to access Pergroup parameters, PersymbolroParams class to access Persymbolro parameters and PerSymbolRwParams class to access Persymbolrw parameters.

Stream classes

When reading output streams containing Decoder, Enhanced Top of Book or Strategy data, use appropriate Stream class to access the data. Decoder class is used to access data from Decoder module, Etob class is used to access data from Enhanced Top of Book module and Strategy class is used to access data from Strategy module.

Pre-Trade Risk Check class

RiskCheckAPI class is used to control Pre-Trade Risk check module. Use methods of this class to enable (or disable) whole module, to set limits of minimal/maximal order price & size or to read counter of dropped orders.

Order Sender class

OrderSenderTop class is used to control Order Sender module. Use methods of this class to enable (or disable) firing orders and to read order sender event messages.
