Magmio::OrderParams::getPergroupSize
unsigned int getPergroupSize(bool include_prepends=true)
Get actual sender pergroup params count in bytes! If include_prepends is set (it is, by default) this count will also contain possible bytes from SID shortcut in FW, which are prepended to regular sender pergroup params! These SID bytes are inaccessible through setPerGroupParam: setPerGroupParam with offset 0 will set the first byte FOLLOWING SID BYTES!
