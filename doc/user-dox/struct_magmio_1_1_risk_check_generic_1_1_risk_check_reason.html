Detailed Description

Bits mapping for "reason" field in TRiskCheckStatus.

A user must cast the raw risk check status (8bit field called "reason") to this structure to map the individual bits to their specific meaning. Ex: Magmio::StatusChannel::TStatusEvents event; Magmio::RiskCheckGeneric::RiskCheckReason *genericReasonMap;

api_ptr->statusChannel.getEvent(event); genericReasonMap = (Magmio::RiskCheckGeneric::RiskCheckReason *) &event.riskCheckEvent.reason; genericReasonMap->priceLow;
