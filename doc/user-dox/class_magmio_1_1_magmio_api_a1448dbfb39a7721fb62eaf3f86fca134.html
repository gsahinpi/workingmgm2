TError Magmio::MagmioApi::setInterfaceScript ( const std::string  start,
const std::string  stop = "",

)

Set paths to scripts controlling configuration of virtual NIC Must be call before API is initialized.

Parameters
startPath to a script that will be called upon initialization of API
stopoptional path to a script that will be called when API is destroyed
ptpTrue to use NIC for Precision Time Protocol. False to set the NIC as a source of data for firmware in card.
Returns
Return SUCCESS on success otherwise return TError
