TError Magmio::OrderSenderApi::send ( const std::string &  session_id,
const unsigned char *  data,
unsigned int  len,
uint8_t  flags = 0 
)

Send packet through the stack.

Parameters
session_idSession ID of connection as entered in configuration file
dataPointer to data to be sent For fix messages: - Data starts with '8=FIX.4.2|9=LLL|35=X|34=NNNNNNNNN|....' (21 bytes before seqnum tag).
  • LLL is body length
  • NNNNNNNNN is sequence number (for KEEP_SW_SEQ_NO, otherwise it's replaced by value from fpga).
  • Payload doesn't end with '10=xxx|'. The checksum tag & value is appended automatically by API.
lenByte length of data
flagsCan be one of following (mutually exclusive): KEEP_SW_SEQ_NO - Keep the application sequence number from the software packet, don't let FPGA to insert its own app seq no.
  • Usage: In certain protocols a message (i.e. heartbeat) is send with the sequence number always set to zero. NOINCR_FPGA_SEQ_NO - After the FPGA inserts its app seq no into the packet, do not increment the value for the next packet.
    • Usage: In certain protocols a message (i.e. heartbeat) doesn't increase the sequence number. Can be combined with: SEND_NOW - Force the packet to go through TCP stack and sent it immediately, without any delays in TCP stack timers
Returns
SUCCESS if no error occured
NOT_CONNECTED if connection was closed
MEMORY_ERROR if there is no free memory in sending buffer
ERROR_TRANSMIT if there are issues sending data
